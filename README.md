# Hand gesture
# 🖐️ Hand Gesture Recognition using Teachable Machine

## Project Overview
A machine learning model that classifies 4 hand gestures in real-time using Google's Teachable Machine platform.

## Classes
| Class | Description |
|-------|-------------|
| Open Hand | All 5 fingers spread wide |
| Fist | All fingers closed |
| Pointing | Index finger extended |
| Partial Close | Transition/ambiguous state |
